 Basic Network Sniffer:

CodeAlpha Cyber Security Internship - Task 1:

Overview:

This project is a Basic Network Sniffer developed using Python and the Scapy library. It captures live network packets and displays useful information about each packet.

Features:

- Captures live network traffic
- Displays Source IP Address
- Displays Destination IP Address
- Detects Protocol (TCP, UDP, ICMP)
- Shows Source and Destination Ports
- Displays Packet Payload
- Runs continuously until stopped

Technologies Used:

- Python 3.13
- Scapy

Installation:

Install Scapy:

"bash
pip install scapy"

 Run

Open Command Prompt as Administrator and run:

"bash
python network_sniffer.py"

 Sample Output:

Source IP : 192.168.0.108
Destination IP : 111.31.99.76
Protocol : TCP
Source Port : 51822
Destination Port : 443
Payload : ...

- Understanding packet capturing
- Understanding TCP/IP protocols
- Basic packet analysis
- Working with Python networking libraries
