# Phishing Awareness Training

# Overview
This project was created as part of the CodeAlpha Cyber Security Internship. It is a presentation designed to educate users about phishing attacks, social engineering techniques, and cybersecurity best practices.

# Topics Covered
- Introduction to Phishing
- Types of Phishing Attacks
- Social Engineering
- Identifying Phishing Emails
- Fake Websites
- Best Practices for Prevention
- Real-World Example
- Phishing Awareness Quiz
