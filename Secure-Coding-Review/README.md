# Secure Coding Review

# Project Description

This project was completed as part of the CodeAlpha Cyber Security Internship. It reviews a Python login application, identifies security vulnerabilities, and implements secure coding improvements.

# Objective:

- Review a Python login program.
- Identify security vulnerabilities.
- Improve the program using secure coding practices.

# Files Included

- "original_login.py" – Original vulnerable Python code.
- "secure_login.py" – Improved secure Python code.
- "Report/Secure_Coding_Review_Report.pdf" – Project report.
- "Screenshots/" – Screenshots of the code and program outputs.

# Security Issues Found

- Hardcoded username and password.
- Weak password.
- Plain-text password storage.
- No limit on login attempts.

# Improvements Made

- Used a stronger password.
- Added a maximum of three login attempts.
- Locked the account after repeated failed logins.

# Technologies Used

- Python 3
- IDLE
